Paragrafos e quebra de linha

Voce pode escrever uma paragrafo de qualquer jeito. Basta colocar tudo no meio do par de tegs <p> e </p>

Caso queira qubrar uma linha
especifica so basta usar a tag <br>

Vamos adicionar alguns simbolos especiais:
® → Representa o simbolo de Marca Registrada
© → Representa o simbolo de Copyright
™ → Representa o simbolo de Trade Mark
€ → Representa o simbolo de Euro
£ → Representa o simbolo de Libra(pound)
¥ → Representa o simbolo de Yen
¢ → Representa o simbolo de Cent
Δ → Representa o simbolo de Delta
↑ → Representa o simbolo de Seta para cima
